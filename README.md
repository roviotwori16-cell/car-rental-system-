Car Rental System  
Overview 
The car rental system simulates a real-world car rental agency where customers can rent and return cars, and the agency 
can manage vehicles and rental transactions.

Features
  Add and manage cars
  Register customers
  Rent and return cars 
  Track car availability
  Calculate rental costs
  

System Entities
1. Car
Represents a vehicle in the system with details such as car ID, model, rental price per day, and availability status.

2. Customer
Represents a customer who can rent cars and also store customer details like ID and name.

3. RentalAgency
Handles all rental operations: managing cars, registering customers, and processing rentals.


Concepts Applied
- Encapsulation: Private attributes with public getters and setters
- Abstraction: Clear method definitions for system operations
- Inheritance: Used where applicable for different car categories
- Polymorphism: Allows flexibility in rental cost calculation
